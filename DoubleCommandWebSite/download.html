Keep up to date

To receive email when a new version is available, subscribe to the mailing list at lists.sourceforge.net/mailman/listinfo/doublecommand-announce.

Get the Source

To get the source code for DoubleCommand, use cvs like so (just press return when asked for a password):

cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/doublecommand login

cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/doublecommand co DoubleCommandSource

cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/doublecommand update -P

For a particular version, use tags like this: DoubleCommand 1.4.1 would be tagged Release-1_4_1, 1.5 would be Release-1_5 and so on. Or browse the source code using the web interface.
